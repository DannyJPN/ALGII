template<class T>
class ADSLibrary::DataStructures::ArrayBasedStructures::DynamicMemory::OOPTemplate::Queue< T >

Fronta implementovaná v poli jako šablona (template) OOP.

Implementace fronty v poli jako kruhový buffer. Implementace v poli znamená, že ukazatel je v tomto případě realizován jako číslo typu int a ukazatel je tudíž běžný index v poli. Velikost fronty lze specifikovat v konstruktoru.
