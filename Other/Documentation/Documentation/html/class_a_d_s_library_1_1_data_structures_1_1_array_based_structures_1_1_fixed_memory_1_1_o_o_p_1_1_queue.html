Detailní popis

Fronta pevné velikosti implementovaná v poli s využitím OOP.

Implementace fronty v poli jako kruhový buffer. Implementace v poli znamená, že ukazatel je v tomto případě realizován jako číslo typu int a ukazatel je tudíž běžný index v poli.
